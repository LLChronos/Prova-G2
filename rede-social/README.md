## Como rodar
1. Entre na pasta do projeto:
   ```bash
   cd rede-social
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Funcionalidades implementadas
- Criação de conta e login
- Logout
- Feed com posts
- Publicação de novos posts
- Curtida/descurtida de posts
- Tema claro/escuro
- Persistência local com localStorage para testes do frontend
